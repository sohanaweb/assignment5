Question-2: What is the difference between innerHTML, innerText, and textContent?

The innerText property sets or returns the text content of an element..The innerHTML property sets or returns the HTML content of an element.

Question-3: What is event delegation in the DOM?

Event delegation in the DOM is a JavaScript technique where you attach a single event listener to a parent element, allowing it to handle events triggered on its child elements.
